6. Linux User Data Transfer
  find /home/usersdata/ -type f -user james -exec cp --parents {} /beta \;
  /home/usersdata/: The starting point for the search.

  -type f: Filters for files only (ignoring directories).
  -user james: Filters for files owned by the user "james".
  -exec ... \;: Runs the following command on every file found.
  cp --parents {} /beta: The {} is a placeholder for the filename found. 
  The --parents flag recreates the source path inside the destination /beta
